HOWTO

== get data ==
Get the scraped list of ideas from scraper wiki http://scraperwiki.com/scrapers/nyc_bigapps_ideas_2/
Export as csv
Clean up in Google Refine

== prepare index ==
Add LaTeX index commands
Make the indexed file into html with tth

== layout ==
Drop body html into template
Copy index table from tth and clean it up
